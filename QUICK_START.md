# Быстрый старт

## Установка и запуск

### 1. Установка зависимостей

Зависимости устанавливаются автоматически при `npm install` в корне проекта:

```bash
# Из корня проекта
npm install
```

Это установит зависимости и для основного проекта, и для микросервиса автоматически (через npm workspaces).

### 2. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 3. Запуск

```bash
# Из корня проекта
npm run dev
```

Откройте `http://localhost:5173/yurservice` - страница будет работать с данными из БД.

## Что работает автоматически

- ✅ Установка зависимостей через npm workspaces
- ✅ Интеграция микросервиса в основное приложение
- ✅ Использование shared кода из `shared-src/` (компоненты, хуки, утилиты)
- ✅ Подключение к БД через переменные окружения
- ✅ Работа в dev и build режимах

## Проверка работы

1. Запустите `npm run dev` из корня проекта
2. Откройте `http://localhost:5173/yurservice`
3. Должна открыться страница "Юр сервисы" с данными из БД
